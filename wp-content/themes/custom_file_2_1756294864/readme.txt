== Description ==

Lifeline Hospital is a versatile WordPress theme designed for hospitals, clinics, and healthcare professionals. The theme features a clean, modern design, making it easy to create a professional online presence. The free version includes essential features like responsive design for seamless browsing on any device, customizable layouts to showcase your services, and a blog section for sharing health tips and updates. For those seeking more advanced functionality, the premium version adds features such as an integrated appointment booking system, a dynamic team section to highlight medical staff, service listings, SEO optimization, and a testimonials area to build trust with patients. == Installation ==

Download the theme zip file from your account on our website.
Go to Appearance > Themes in your WordPress dashboard.
Click the "Add New" button at the top of the screen, then click "Upload Theme".
Choose the theme zip file and click "Install Now".
Once the theme is installed, click the "Activate" button to activate it.
